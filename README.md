# Student Data Analysis Project

## Description
This project cleans and analyzes Excel data using Python.

## Features
- Removes duplicates
- Handles missing values
- Calculates average marks
- Generates graphs

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## How to Run
1. Install libraries:
   pip install pandas matplotlib openpyxl

2. Run:
   python project.py

## Output
- Cleaned Excel file
- Graph visualization
